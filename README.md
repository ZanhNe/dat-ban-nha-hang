## Mô ta

## Công nghệ sử dụng
- Backend: Spring Boot
- Frontend: ReactJS
- Database: MySQL
- Map: Goong API
## Cài đặt và chạy
### Yêu cầu
- Java 17+
- Node.js 18+ (nếu dùng React)
- MySQL
### Chạy Backend
cd backend/dat-ban-nha-hang
./mvnw spring-boot:run
### Chạy Frontend (nếu dùng React)
cd frontend/dat-ban-nha-hang
npm install
npm run dev

### Truy cập
- Frontend: http://localhost:5173
- Backend API: http://localhost:8080
